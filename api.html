hpye: Client-Server API

This is currently a non-adversarial API. hype_server may currently not correctly handle mal-formed or erroneous messages; it is the client's responsibility to send correct messages or risk server process crashing.

Search Queries

  • search query
  • Returns a (stringified) json list of songid, artist, song pairs.
    • Example: "[['1m9n2', 'Kesha', 'Blow (some remix)'], ['3enz3', 'Ke$ha', 'Take It Off (In da club rmx)']]"

Play Song From Query Result

  • play songid
  • Plays the song with song ID songid, forgoing the play queue.
  • Returns "OK" if successfully parsed query and retrieving song; "ERROR_REMOVED" if the song was removed and thus cannot be played; "ERROR" if there was an error.

Pause / Resume Playback

  • pauseresume
  • Returns OK_PAUSED if the song was paused, or OK_RESUMED if the song was resumed.
